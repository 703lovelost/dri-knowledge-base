## 1. –ü–æ–Ω—è—Ç–∏–µ —Å—Ç–µ–ø–µ–Ω–∏ –≤–µ—Ä—à–∏–Ω—ã –≤ –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –∏ –Ω–µ–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –≥—Ä–∞—Ñ–µ. –¢–µ–æ—Ä–µ–º—ã –æ —Å—Ç–µ–ø–µ–Ω—è—Ö –≤–µ—Ä—à–∏–Ω –≤ –Ω–µ–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –∏ –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –≥—Ä–∞—Ñ–µ.

**–°—Ç–µ–ø–µ–Ω—å –≤–µ—Ä—à–∏–Ω—ã** ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ [–∏–Ω—Ü–∏–¥–µ–Ω—Ç–Ω—ã—Ö](#edge-names) —Ä—ë–±–µ—Ä, —Å—á–∏—Ç–∞—è [–ø–µ—Ç–ª–∏](#pseudograph) –¥–≤–∞–∂–¥—ã. –û–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è –∫–∞–∫ $deg(v)$

–í–µ—Ä—à–∏–Ω–∞ —Å–æ —Å—Ç–µ–ø–µ–Ω—å—é 0 ‚Äî **–∏–∑–æ–ª–∏—Ä–æ–≤–∞–Ω–Ω–∞—è**.

<a name="pendant-vertex"></a>–í–µ—Ä—à–∏–Ω–∞ —Å–æ —Å—Ç–µ–ø–µ–Ω—å—é 1 ‚Äî **–≤–∏—Å—è—á–∞—è**

–î–ª—è [–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ](#directed-graph) –≥—Ä–∞—Ñ–∞ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω—ã –ø–æ–Ω—è—Ç–∏—è **–ø–æ–ª—É—Å—Ç–µ–ø–µ–Ω—å –∑–∞—Ö–æ–¥–∞** $deg^-(v)$ (–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä—ë–±–µ—Ä –¥–ª—è –∫–æ—Ç–æ—Ä—ã—Ö –≤–µ—Ä—à–∏–Ω–∞ ‚Äî [–∫–æ–Ω–µ—á–Ω–∞—è](#vertex-names)) –∏ **–ø–æ–ª—É—Å—Ç–µ–ø–µ–Ω—å –∏—Å—Ö–æ–¥–∞** $deg^+(v)$ (–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–µ–±–µ—Ä –¥–ª—è –∫–æ—Ç–æ—Ä—ã—Ö –≤–µ—Ä—à–∏–Ω–∞ ‚Äî [–Ω–∞—á–∞–ª—å–Ω–∞—è](#vertex-names)).


üí° –¢–µ–æ—Ä–µ–º–∞ (**–û —Ä—É–∫–æ–ø–æ–∂–∞—Ç–∏–∏**):
–í –Ω–µ–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –≥—Ä–∞—Ñ–µ, –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–µ–±–µ—Ä, —É–º–Ω–æ–∂–µ–Ω–Ω–æ–µ –Ω–∞ –¥–≤–∞, —Ä–∞–≤–Ω–æ —Å—É–º–º–µ —Å—Ç–µ–ø–µ–Ω–µ–π –≤—Å–µ—Ö –≤–µ—Ä—à–∏–Ω.

$2m = \sum\limits_{v\in V}deg(v)$

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:
–ö–∞–∂–¥–æ–µ —Ä–µ–±—Ä–æ –ø—Ä–∏–±–∞–≤–ª—è–µ—Ç 2 –∫ —Å—É–º–º–µ —Å—Ç–µ–ø–µ–Ω–µ–π, —Ç–∞–∫ –∫–∞–∫ –∏–Ω—Ü–∏–¥–µ–Ω—Ç–Ω–æ –ª–∏–±–æ –¥–≤—É–º –≤–µ—Ä—à–∏–Ω–∞–º, –ª–∏–±–æ —è–≤–ª—è–µ—Ç—Å—è –ø–µ—Ç–ª—ë–π –∏ –≤—Å—ë —Ä–∞–≤–Ω–æ –ø—Ä–∏–≤–Ω–æ—Å–∏—Ç 2.




üí° –¢–µ–æ—Ä–µ–º–∞ (**–û –≤–µ—Ä—à–∏–Ω–∞—Ö –Ω–µ—á—ë—Ç–Ω–æ–π —Å—Ç–µ–ø–µ–Ω–∏**):
–í –Ω–µ–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –≥—Ä–∞—Ñ–µ, –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤–µ—Ä—à–∏–Ω –Ω–µ—á—ë—Ç–Ω–æ–π —Å—Ç–µ–ø–µ–Ω–∏ —á—ë—Ç–Ω–æ.

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:
–ü—É—Å—Ç—å $V_e$ ‚Äî –º–Ω–æ–∂–µ—Å—Ç–≤–æ –≤–µ—Ä—à–∏–Ω —á–µ—Ç–Ω–æ–π —Å—Ç–µ–ø–µ–Ω–∏, –∞ $V_o$ ‚Äî –Ω–µ—á—ë—Ç–Ω–æ–π.

\sum\limits_{v\in V_e} deg(v) +

```python
     queue = Queue()
     explored[root] = True
     queue.enqueue(root)
     while not queue.isEmpty():
        v = queue.dequeue()
        for (v,u) in G.edgesFrom(v):
             if not explored[u]:
                  explored[u] = True
                  parent[u] = v
									# color[u] = Gray
                  queue.enqueue(u)
				# color[u] = Black
```

```python
def DFS(G, root):
     stack = Stack()
     explored[root] = True
     stack.push(root)
     while not stack.isEmpty():
        v = stack.pop()
        for (v,u) in G.edgesFrom(v):
             if not explored[u]:
                  explored[u] = True
                  parent[u] = v
									# color[u] = Gray
                  stack.push(u)
				# color[u] = Black
```

## 11. –õ–µ–º–º–∞ –æ —Ç–æ–º, –≤ –∫–∞–∫–∏—Ö —Å–ª—É—á–∞—è—Ö –≤–µ—Ä—à–∏–Ω–∞ —è–≤–ª—è–µ—Ç—Å—è —Ç–æ—á–∫–æ–π —Å–æ—á–ª–µ–Ω–µ–Ω–∏—è –≤ –¥–µ—Ä–µ–≤–µ –ø–æ–∏—Å–∫–∞ –≤ –≥–ª—É–±–∏–Ω—É

1) –ü–æ–∫–∞–∂–µ–º, —á—Ç–æ –∫–æ—Ä–µ–Ω—å —è–≤–ª—è–µ—Ç—Å—è —Ç–æ—á–∫–æ–π —Å–æ—á–ª–µ–Ω–µ–Ω–∏—è $\iff$—É –Ω–µ–≥–æ $>1$ —Å—ã–Ω–æ–≤–µ–π.

$\Rightarrow$ –ü—Ä–∏ —É–¥–∞–ª–µ–Ω–∏–∏ –∫–æ—Ä–Ω—è —Å –æ–¥–Ω–∏–º —Å—ã–Ω–æ–º –∏–ª–∏ –±–µ–∑ —Å—ã–Ω–æ–≤–µ–π –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç —Å–≤—è–∑–Ω–æ—Å—Ç–∏ –Ω–µ –º–æ–∂–µ—Ç —É–≤–µ–ª–∏—á–∏—Ç—å—Å—è.

$\Leftarrow$ –ï—Å–ª–∏ —É –∫–æ—Ä–Ω—è –±–æ–ª—å—à–µ 1 —Å—ã–Ω–∞, —Ç–æ –ø—Ä–∏ –µ–≥–æ —É–¥–∞–ª–µ–Ω–∏–∏, –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç —Å–≤—è–∑–Ω–æ—Å—Ç–∏ —É–≤–µ–ª–∏—á–∏—Ç—Å—è, —Ç–∞–∫ –∫–∞–∫ –∫–∞–∂–¥–∞—è ‚Äú–≤–µ—Ç–≤—å‚Äù –¥–µ—Ä–µ–≤–∞ –æ–±—Ä–∞–∑—É–µ—Ç –Ω–æ–≤—É—é (–ø–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—é –¥–µ—Ä–µ–≤–∞, –≤–µ–¥—å –æ–±—Ä–∞—Ç–Ω—ã—Ö —Ä—ë–±–µ—Ä –Ω–µ –±—É–¥–µ—Ç –ø—Ä–∏ —É–¥–∞–ª–µ–Ω–∏–∏ –∫–æ—Ä–Ω—è). 

2) –ü–æ–∫–∞–∂–µ–º, —á—Ç–æ *–Ω–µ* –∫–æ—Ä–µ–Ω—å —è–≤–ª—è–µ—Ç—Å—è —Ç–æ—á–∫–æ–π —Å–æ—á–ª–µ–Ω–µ–Ω–∏—è $\iff$ —É $s$ ‚Äî *–Ω–µ–∫–æ—Ç–æ—Ä–æ–≥–æ* —Ä–µ–±—ë–Ω–∫–∞ $a$ –Ω–µ—Ç –æ–±—Ä–∞—Ç–Ω—ã—Ö —Ä–µ–±–µ—Ä *–Ω–∏ –∏–∑ –æ–¥–Ω–æ–≥–æ* –∏–∑ –ø–æ—Ç–æ–º–∫–æ–≤ (–≤–∫–ª—é—á–∞—è —Å–∞–º—É $s$) *–Ω–∏ –≤ –æ–¥–Ω–æ–≥–æ* –ø—Ä–µ–¥–∫–∞ $a$

$\Leftarrow$  –ü—É—Å—Ç—å $p$ ‚Äî —Ä–æ–¥–∏—Ç–µ–ª—å $a$. –ö–∞–∂–¥–æ–µ –æ–±—Ä–∞—Ç–Ω–æ–µ —Ä–µ–±—Ä–æ –∏–∑ $v$ ‚Äî –ø–æ—Ç–æ–º–∫–∞ $s$ –∏–¥—ë—Ç –≤ –ø—Ä–µ–¥–∫–∞ $v$. –í –ø—Ä–µ–¥–∫–∞ $a$ —Ä–µ–±—Ä–æ –ø–æ –ø—Ä–µ–¥–ø–æ–ª–æ–∂–µ–Ω–∏—é –∏–¥—Ç–∏ –Ω–µ –º–æ–∂–µ—Ç. –ó–Ω–∞—á–∏—Ç –æ–Ω–æ –∏–¥–µ—Ç –ª–∏–±–æ –≤ $a$, –ª–∏–±–æ –≤ –µ–≥–æ –ø–æ—Ç–æ–º–∫–∞. –ê –∑–Ω–∞—á–∏—Ç, –ª—é–±–æ–π –ø—É—Ç—å –∏–∑ $s$ –¥–æ $p$ –ø—Ä–æ—Ö–æ–¥–∏—Ç —á–µ—Ä–µ–∑ $a$. –ó–Ω–∞—á–∏—Ç $a$ ‚Äî —Ç–æ—á–∫–∞ —Å–æ—á–ª–µ–Ω–µ–Ω–∏—è.

$\Rightarrow$ –ü—É—Å—Ç—å $x,y$ ‚Äî —Ä–∞–∑–ª–∏—á–Ω—ã–µ –≤–µ—Ä—à–∏–Ω—ã, –æ—Ç–ª–∏—á–Ω—ã–µ –æ—Ç $a$, —Ç–∞–∫–∏–µ —á—Ç–æ –ª—é–±–æ–π –ø—É—Ç—å –º–µ–∂–¥—É $x$ –∏ $y$ –ø—Ä–æ—Ö–æ–¥–∏—Ç —á–µ—Ä–µ–∑ $a$. –ü–æ –º–µ–Ω—å—à–µ–π –º–µ—Ä–µ –æ–¥–Ω–∞ –∏–∑ –Ω–∏—Ö, –ø–æ–ª–æ–∂–∏–º $x$ –∏–Ω–∞—á–µ –ø–µ—Ä–µ–∏–º–µ–Ω—É–µ–º, —è–≤–ª—è–µ—Ç—Å—è –ø–æ—Ç–æ–º–∫–æ–º $a$. (–∏–Ω–∞—á–µ –º–µ–∂–¥—É –Ω–∏–º–∏ –µ—Å—Ç—å –ø—É—Ç—å –ø–æ —Ä–µ–±—Ä–∞–º –¥–µ—Ä–µ–≤–∞, –º–∏–Ω—É—é—â–∏–π $a$) –ü—É—Å—Ç—å $s$ ‚Äî —Å—ã–Ω $a$, —Ç–∞–∫–æ–π —á—Ç–æ $x$ ‚Äî –ø–æ—Ç–æ–º–æ–∫ $s$ (–≤–æ–∑–º–æ–∂–Ω–æ $x=s$). –¢–æ–≥–¥–∞ –ª–∏–±–æ –Ω–µ—Ç –æ–±—Ä–∞—Ç–Ω–æ–≥–æ —Ä–µ–±—Ä–∞ –æ—Ç $v$ ‚Äî –ø–æ—Ç–æ–º–∫–∞ $s$ –∫ $w$ ‚Äî –ø—Ä–µ–¥–∫—É $a$ (—Ç–æ–≥–¥–∞ —É—Å–ª–æ–≤–∏–µ 2 –≤—ã–ø–æ–ª–Ω–µ–Ω–æ), –ª–∏–±–æ —Ç–∞–∫–æ–µ —Ä–µ–±—Ä–æ $(v,w)$ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç. –¢–æ–≥–¥–∞ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –¥–≤–∞ —Å–ª—É—á–∞—è:

1.  –ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, $y$ ‚Äî –Ω–µ –ø–æ—Ç–æ–º–æ–∫ $a$. –¢–æ–≥–¥–∞ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –ø—É—Ç—å –∏–∑ $x$ —á–µ—Ä–µ–∑ $v$ –≤ $y$, –Ω–µ —Å–æ–¥–µ—Ä–∂–∞—â–∏–π $a$ ‚Äî –ø—Ä–æ—Ç–∏–≤–æ—Ä–µ—á–∏–µ —Å —É—Å–ª–æ–≤–∏–µ–º

2. –ó–Ω–∞—á–∏—Ç, $y$ ‚Äî –ø–æ—Ç–æ–º–æ–∫ $a$. $y$ ‚Äî –Ω–µ –ø–æ—Ç–æ–º–æ–∫ $s$, –∏–Ω–∞—á–µ –±—ã —Å—É—â–µ—Å—Ç–≤–æ–≤–∞–ª –ø—É—Ç—å –∏–∑ $x$ –≤ $y$, –Ω–µ —Å–æ–¥–µ—Ä–∂–∞—â–∏–π $a$. –ü—É—Å—Ç—å $s'$ ‚Äî —Å—ã–Ω $a$, —Ç–∞–∫–æ–π —á—Ç–æ $y$ ‚Äî –ø–æ—Ç–æ–º–æ–∫ $s'$. –¢–æ–≥–¥–∞ –ª–∏–±–æ –Ω–µ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –æ–±—Ä–∞—Ç–Ω–æ–≥–æ —Ä–µ–±—Ä–∞ –º–µ–∂–¥—É $v'$ ‚Äî –ø–æ—Ç–æ–º–∫–æ–º $s'$ –∏ $w'$ ‚Äî –ø—Ä–µ–¥–∫–æ–º $a$ (—Ç–æ–≥–¥–∞ —É—Å–ª–æ–≤–∏–µ 2 –≤—ã–ø–æ–ª–Ω–µ–Ω–æ), –ª–∏–±–æ —Ç–∞–∫–æ–µ —Ä–µ–±—Ä–æ $(v',w')$ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç. –¢–æ–≥–¥–∞ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –ø—É—Ç—å –æ—Ç $x$ –∫ $v$ –∫ $w$ –∫ $w'$ –∫ $v'$ –∫ $y$, –Ω–µ —Å–æ–¥–µ—Ä–∂–∞—â–∏–π $a$ ‚Äî–ø—Ä–æ—Ç–∏–≤–æ—Ä–µ—á–∏–µ —Å —É—Å–ª–æ–≤–∏–µ–º.

## 12. $st$-–Ω—É–º–µ—Ä–∞—Ü–∏—è. –ê–ª–≥–æ—Ä–∏—Ç–º –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏—è $st$-–Ω—É–º–µ—Ä–∞—Ü–∏–∏.

**$st$-–Ω—É–º–µ—Ä–∞—Ü–∏—è** (**–±–∏–ø–æ–ª—è—Ä–Ω–∞—è –Ω—É–º–µ—Ä–∞—Ü–∏—è**) ‚Äî —Å–ø–æ—Å–æ–± –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–∏—è —Ä—ë–±–µ—Ä –Ω–µ–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ [–¥–≤—É—Å–≤—è–∑–Ω–æ–≥–æ](#bipartite-graph) –≥—Ä–∞—Ñ–∞, —Ç–∞–∫–æ–π —á—Ç–æ:

- –°—É—â–µ—Å—Ç–≤—É–µ—Ç –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω—ã–π –∏—Å—Ç–æ—á–Ω–∏–∫ $s$ –∏ –ø—Ä–∏—ë–º–Ω–∏–∫ $t$. (–¢–æ–ª—å–∫–æ –∏—Å—Ö–æ–¥—è—â–∏–µ –∏ —Ç–æ–ª—å–∫–æ –≤—Ö–æ–¥—è—â–∏–µ —Ä—ë–±—Ä–∞ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ)
- –ù–µ—Ç —Ü–∏–∫–ª–æ–≤

$Low(v) = \min(\{v\}\cup\{w\})$, –≥–¥–µ $x$ ‚Äî –ø–æ—Ç–æ–º–æ–∫ $v$, $w$ ‚Äî –ø—Ä–µ–¥–æ–∫ $v$, –∏ —Å—É—â–µ—Å—Ç–≤–µ—Ç —Ä–µ–±—Ä–æ $(x,w)$. 

–ê–ª–≥–æ—Ä–∏—Ç–º –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏—è $st$-–Ω—É–º–µ—Ä–∞—Ü–∏–∏:

- 1 –ø—Ä–æ—Ö–æ–¥ ‚Äî DFS.
–í–µ—Ä—à–∏–Ω—É 1 –Ω–∞–∑–Ω–∞—á–∞–µ–º $s$, –≤–µ—Ä—à–∏–Ω—É 2 ‚Äî  $t$.
–î–ª—è –∫–∞–∂–¥–æ–π –≤–µ—Ä—à–∏–Ω—ã –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è:
    - $v$ ‚Äî –Ω–æ–º–µ—Ä –≤–µ—Ä—à–∏–Ω—ã –≤ DFS
    - $Low(v)$
    - $p(v)$ ‚Äî –ø—Ä–µ–¥–æ–∫
- 2 –ø—Ä–æ—Ö–æ–¥ (–≤ –ø–æ—Ä—è–¥–∫–µ –Ω–æ–º–µ—Ä–æ–≤ –≤–µ—Ä—à–∏–Ω –≤ DFS)

```python
L = {s,t}
sign(s) = '-'

for v in V\{s,t}:
	if sign(low(v)) == '+':
		L.insert(v, after, p(v))
		sign(p(v)) = '-'
	if sign(low(v)) == '-':
		L.insert(v, before, p(v))
		sign(p(v)) = '+'
```

## 14. –ü–æ–Ω—è—Ç–∏–µ –≤–µ—Ä—à–∏–Ω–Ω–æ–π —Å–≤—è–∑–Ω–æ—Å—Ç–∏, —Ä–µ–±–µ—Ä–Ω–æ–π —Å–≤—è–∑–Ω–æ—Å—Ç–∏, —Å–≤—è–∑—å –º–µ–∂–¥—É –≤–µ—Ä—à–∏–Ω–Ω–æ–π —Å–≤—è–∑–Ω–æ—Å—Ç—å—é, —Ä–µ–±–µ—Ä–Ω–æ–π —Å–≤—è–∑–Ω–æ—Å—Ç—å—é –∏ –º–∏–Ω–∏–º–∞–ª—å–Ω–æ–π —Å—Ç–µ–ø–µ–Ω—å—é –≤–µ—Ä—à–∏–Ω—ã –≤ –≥—Ä–∞—Ñ–µ.

–ú—ã –≥–æ–≤–æ—Ä–∏–º —á—Ç–æ –≥—Ä–∞—Ñ **$k$-—Å–≤—è–∑–µ–Ω**, –µ—Å–ª–∏ $\kappa(G) > k$.

**–†–∞–∑—Ä–µ–∑** ‚Äî –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ *—Ä–µ–±–µ—Ä* –≥—Ä–∞—Ñ–∞, –ø—Ä–∏ —É–¥–∞–ª–µ–Ω–∏–∏ –∫–æ—Ç–æ—Ä—ã—Ö, –æ–Ω —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è –Ω–µ—Å–≤—è–∑–Ω—ã–º.

**–ß–∏—Å–ª–æ —Ä—ë–±–µ—Ä–Ω–æ–π —Å–≤—è–∑–Ω–æ—Å—Ç–∏** ‚Äî  –º–∏–Ω–∏–º–∞–ª—å–Ω–æ–µ —á–∏—Å–ª–æ —Ä—ë–±–µ—Ä –≤ —Ä–∞–∑—Ä–µ–∑–µ –≥—Ä–∞—Ñ–∞. –û–±–æ–∑–Ω. $\lambda(G)$.

$\Rightarrow$ –°—É—â–µ—Å—Ç–≤—É–µ—Ç 2 —Ä–∞–∑–ª–∏—á–Ω—ã–µ $v,w$, –æ—Ç–ª–∏—á–Ω—ã–µ –æ—Ç $a$, —Ç–∞–∫–∏–µ —á—Ç–æ –ª—é–±–æ–π –ø—É—Ç—å –º–µ–∂–¥—É $v$ –∏ $w$ —Å–æ–¥–µ—Ä–∂–∏—Ç $a$. –¢–∞–∫ –∫–∞–∫ $G$ —Å–≤—è–∑–µ–Ω, —Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω —Ç–∞–∫–æ–π –ø—É—Ç—å —Å—É—â–µ—Å—Ç–≤—É–µ—Ç. –ü—É—Å—Ç—å $(x,a),(y,a)$ ‚Äî —Ä—ë–±—Ä–∞ –Ω–∞ —ç—Ç–æ–º –ø—É—Ç–∏. –ï—Å–ª–∏ –±—ã –±—ã–ª —Ü–∏–∫–ª, —Å–æ–¥–µ—Ä–∂–∞—â–∏–π –¥–≤–∞ —ç—Ç–∏—Ö —Ä–µ–±—Ä–∞, –∑–Ω–∞—á–∏—Ç —Å—É—â–µ—Å—Ç–≤–æ–≤–∞–ª –ø—É—Ç—å –º–µ–∂–¥—É $v$ –∏ $w$, –Ω–µ —Å–æ–¥–µ—Ä–∂–∞—â–∏–π $a$. –ó–Ω–∞—á–∏—Ç, $(x,a),(y,a)$ –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤ —Ä–∞–∑–Ω—ã—Ö –¥–≤—É—Å–≤—è–∑–Ω—ã—Ö –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞—Ö –∏ $a$ –≤ –∏—Ö –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–∏

$\Leftarrow$ –ï—Å–ª–∏ $a\in V_i\cap V_j$, –∑–Ω–∞—á–∏—Ç –µ—Å—Ç—å —Ä—ë–±—Ä–∞ $(x,a), (y,a)$ –≤ $E_i$ –∏ $E_j$ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ. –¢–∞–∫ –∫–∞–∫ –æ–Ω–∏ –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–∞—Ç —Ä–∞–∑–Ω—ã–º –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞–º —Å–≤—è–∑–Ω–æ—Å—Ç–∏, –æ–Ω–∏ –Ω–µ –º–æ–≥—É—Ç –ª–µ–∂–∞—Ç—å –≤ –æ–¥–Ω–æ–º —Ü–∏–∫–ª–µ, –∞ –∑–Ω–∞—á–∏—Ç –ª—é–±–æ–π –ø—É—Ç—å –æ—Ç $x$  –¥–æ $y$ —Å–æ–¥–µ—Ä–∂–∏—Ç $a$.



## 16. –ê–ª–≥–æ—Ä–∏—Ç–º –î–µ–π–∫—Å—Ç—Ä—ã –ø–æ–∏—Å–∫–∞ –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –≤ –≥—Ä–∞—Ñ–µ. –ö–æ—Ä—Ä–µ–∫—Ç–Ω–æ—Å—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –î–µ–π–∫—Å—Ç—Ä—ã

–ê–ª–≥–æ—Ä–∏—Ç–º **–î–µ–π–∫—Å—Ç—Ä—ã** –Ω–∞—Ö–æ–¥–∏—Ç –º–∏–Ω–∏–º–∞–ª—å–Ω–æ–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ *–∏–∑ –æ–¥–Ω–æ–π –≤–µ—Ä—à–∏–Ω—ã* –¥–ª—è –≤–∑–≤–µ—à–µ–Ω–Ω–æ–≥–æ, –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –≥—Ä–∞—Ñ–∞ –≤ —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ *–≤—Å–µ –≤–µ—Å–∞ –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–µ*.

```python
distance = [inf for _ in range(|V|)]
previous = [-1 for _ in range(|V|)]
distance[start] = 0;

queue = PriorityQueue()
queue.push(start, distance[start])

while (!queue.isEmpty()):
	vertex = queue.pop()
	if isVisited[vertex]:
		continue
	isVisited[vertex] = True
	for (to, edgeWeight) in adjacents[vertex]:
		if distance[to] > distance[vertex] + edgeWeight:
			dist[to] = distance[vertex] + edgeWeight
			previous[to] = vertex
			queue.push(to, distance[to])
```

–°–ª–æ–∂–Ω–æ—Å—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º–∞ ‚Äî $O(V^2)$.


üí° –¢–µ–æ—Ä–µ–º–∞ (**–û –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ—Å—Ç–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –î–µ–π–∫—Å—Ç—Ä—ã**):
–ê–ª–≥–æ—Ä–∏—Ç–º –î–µ–π–∫—Å—Ç—Ä—ã –∑–∞–ø—É—â–µ–Ω–Ω—ã–π –Ω–∞ –≤–∑–≤–µ—à–µ–Ω–Ω–æ–º, –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –≥—Ä–∞—Ñ–µ —Å –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–π –≤–µ—Å–æ–≤–æ–π —Ñ—É–Ω–∫—Ü–∏–µ–π $w$ –∏–∑ –∏—Å—Ç–æ—á–Ω–∏–∫–∞ $s$ –æ–∫–∞–Ω—á–∏–≤–∞–µ—Ç—Å—è —Å $\forall u\in V: u.d = \delta (s,u)$

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ ‚Äî –∏–Ω–¥—É–∫—Ü–∏—è –ø–æ –æ–±—Ä–∞–±–æ—Ç–∞–Ω–Ω—ã–º –≤–µ—Ä—à–∏–Ω–∞–º



## 17. –ê–ª–≥–æ—Ä–∏—Ç–º –ë–µ–ª–ª–º–∞–Ω–∞-–§–æ—Ä–¥–∞ –ø–æ–∏—Å–∫–∞ –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –≤ –≥—Ä–∞—Ñ–µ

–ê–ª–≥–æ—Ä–∏—Ç–º **–ë–µ–ª–ª–º–∞–Ω–∞-–§–æ—Ä–¥–∞** –Ω–∞—Ö–æ–¥–∏—Ç –∫—Ä–∞—Ç—á–∞–π—à–∏–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è *–æ—Ç –æ–¥–Ω–æ–π –≤–µ—Ä—à–∏–Ω—ã* –¥–æ –æ—Å—Ç–∞–ª—å–Ω—ã—Ö. –†–∞–±–æ—Ç–∞–µ—Ç –≤ —Å–ª—É—á–∞–µ —Ä—ë–±–µ—Ä –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–≥–æ –≤–µ—Å–∞.

```python
for v in V:
	for pathLen in range(n - 1):
		dist[v][pathLen] = inf

dist[start][0] = 0

for i in range(n - 1):
	for (u,v) in E:
		if dist[v][i] > dist[v][i-1] + weight(u,v):
			dist[v][i] = dist[v][i-1] + weight(u,v)
			prev[v][i] = u
```

–ü—Ä–∏ –¥–æ–±–∞–≤–ª–µ–Ω–∏–∏ –µ—â—ë –æ–¥–Ω–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏ –∏ –ø—Ä–æ–≤–µ—Ä–∫–∏ –Ω–∞ –∏–∑–º–µ–Ω–µ–Ω–∏—è, –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ–±–Ω–∞—Ä—É–∂–∏—Ç—å –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–µ —Ü–∏–∫–ª—ã, –¥–æ—Å—Ç–∏–∂–∏–º—ã–µ –∏–∑ —Å—Ç–∞—Ä—Ç–æ–≤–æ–π –≤–µ—Ä—à–∏–Ω—ã.

## 18. –ê–ª–≥–æ—Ä–∏—Ç–º—ã –ø–æ–∏—Å–∫–∞ –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –º–µ–∂–¥—É –≤—Å–µ–º–∏ –≤–µ—Ä—à–∏–Ω–∞–º–∏ –≥—Ä–∞—Ñ–∞ –Ω–∞ –æ—Å–Ω–æ–≤–µ –º–∞—Ç—Ä–∏—á–Ω–æ–≥–æ —É–º–Ω–æ–∂–µ–Ω–∏—è

–ú–æ–∂–Ω–æ –Ω–∞–π—Ç–∏ –¥–ª–∏–Ω—É –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –¥–ª–∏–Ω—ã –Ω–µ –±–æ–ª–µ–µ $t$ –º–µ–∂–¥—É –≤–µ—Ä—à–∏–Ω–∞–º–∏, –≤–æ–∑–≤–µ–¥—è –º–∞—Ç—Ä–∏—Ü—É —Å–º–µ–∂–Ω–æ—Å—Ç–∏ –≥—Ä–∞—Ñ–∞ –≤ —Å—Ç–µ–ø–µ–Ω—å $t$, —Å–æ —Å–ª–µ–¥—É—é—â–∏–º ‚Äú—É–º–Ω–æ–∂–µ–Ω–∏–µ–º‚Äù: $C_{ij} = \min\limits_{p=1\dots n}(A_{ip} + B_{pj})$

## 19. –ê–ª–≥–æ—Ä–∏—Ç–º –§–ª–æ–π–¥–∞-–£–æ—Ä—à–µ–ª–ª–∞

–ê–ª–≥–æ—Ä–∏—Ç–º **–§–ª–æ–π–¥–∞-–£–æ—Ä—à–µ–ª–ª–∞** –Ω–∞—Ö–æ–¥–∏—Ç –∫—Ä–∞—Ç—á–∞–π—à–∏–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è *–º–µ–∂–¥—É –≤—Å–µ–º–∏ –ø–∞—Ä–∞–º–∏ –≤–µ—Ä—à–∏–Ω*. –†–∞–±–æ—Ç–∞–µ—Ç –≤ —Å–ª—É—á–∞–µ —Ä—ë–±–µ—Ä –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–≥–æ –≤–µ—Å–∞.

```python
dist = [[inf * n] * n]

for (u, v) in E
    dist[u][v] = weight(u, v)
for each vertex v do
    dist[v][v] = 0

for k in range(n)
    for i from range(n)
        for j from range(n)
            if dist[i][j] > dist[i][k] + dist[k][j]:
                dist[i][j] = dist[i][k] + dist[k][j]
```

## 20. –ê–ª–≥–æ—Ä–∏—Ç–º –î–∂–æ–Ω—Å–æ–Ω–∞

üí° –õ–µ–º–º–∞ (**–û —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏–∏ –∫—Ä–∞—Ç—á–∞–π—à–∏—Ö –ø—É—Ç–µ–π –ø—Ä–∏ –∏–∑–º–µ–Ω–µ–Ω–∏–∏ –≤–µ—Å–æ–≤**):
–û–ø—Ä–µ–¥–µ–ª–∏–º –Ω–æ–≤—É—é –≤–µ—Å–æ–≤—É—é —Ñ—É–Ω–∫—Ü–∏—é: $w'(u,v) = w(u,v) + h(u) - h(v)$,
–≥–¥–µ $h: V\to \R$ ‚Äî –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–µ —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –≤–µ—Ä—à–∏–Ω —á–∏—Å–ª–∞–º.

–ü—É—Å—Ç—å $p=(0,1,\dots,k)$ ‚Äî –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—ã–π –ø—É—Ç—å –æ—Ç –≤–µ—Ä—à–∏–Ω—ã $0$ –¥–æ –≤–µ—Ä—à–∏–Ω—ã $k$.
–¢–æ–≥–¥–∞ $p$ ‚Äî –∫—Ä–∞—Ç—á–∞–π—à–∏–π –ø—É—Ç—å –æ—Ç $0$ –¥–æ $k$ —Å –≤–µ—Å–æ–≤–æ–π —Ñ—É–Ω–∫—Ü–∏–µ–π $w$$\iff$
$p$ ‚Äî –∫—Ä–∞—Ç—á–∞–π—à–∏–π –ø—É—Ç—å –æ—Ç $0$ –¥–æ $k$ —Å –≤–µ—Å–æ–≤–æ–π —Ñ—É–Ω–∫—Ü–∏–µ–π $w'$.

–¢–∞–∫ –∂–µ, $G$ –∏–º–µ–µ—Ç —Ü–∏–∫–ª –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–≥–æ –≤–µ—Å–∞ —Å $w$ $\iff$
$G$ –∏–º–µ–µ—Ç —Ü–∏–∫–ª –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–≥–æ –≤–µ—Å–∞ —Å $w'$



–¢–µ–ø–µ—Ä—å —Å–¥–µ–ª–∞–µ–º –≤–µ—Å–æ–≤—É—é —Ñ—É–Ω–∫—Ü–∏—é –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–π:

- –î–æ–±–∞–≤–∏–º –≤–µ—Ä—à–∏–Ω—É $s$, —Å–º–µ–∂–Ω—É—é —Å–æ –≤—Å–µ–º–∏
- –†–∞—Å—à–∏—Ä–∏–º –≤–µ—Å–æ–≤—É—é —Ñ—É–Ω–∫—Ü–∏—é, —Ç–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —á—Ç–æ $\forall v\in V:w(s,v) = 0$
- –û–ø—Ä–µ–¥–µ–ª–∏–º $h(v) = \delta(s,v)$ –∏ –∏—Å–ø–æ–ª—å–∑—É–µ–º –∫–∞–∫ –≤ –ª–µ–º–º–µ –≤—ã—à–µ

```python
		dist[u] = [d - h[u] + h[v] for d in run_dijkstra(G, new_weight, u)]
```

–û–ø—Ä–µ–¥–µ–ª–∏–º $bottleneck(P,f)$ –∫–∞–∫ –º–∏–Ω–∏–º–∞–ª—å–Ω—É—é –æ—Å—Ç–∞—Ç–æ—á–Ω—É—é –ø—Ä–æ–ø—É—Å–∫–Ω—É—é —Å–ø–æ—Å–æ–±–Ω–æ—Å—Ç—å —Ä–µ–±—Ä–∞ –∏–∑ $P$ –ø—Ä–∏ –ø–æ—Ç–æ–∫–µ $f$.

–û–ø—Ä–µ–¥–µ–ª–∏–º –æ–ø–µ—Ä–∞—Ü–∏—é $augment(f,P)$, —Å–æ–∑–¥–∞—é—â—É—é –Ω–æ–≤—ã–π –ø–æ—Ç–æ–∫ –≤ —Å–µ—Ç–∏:

```python
def augment(f, P):

	for (u,v) in P:

		else:
			f(e) -= bottleneck
	return f
```
