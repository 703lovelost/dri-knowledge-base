### <a name="1-14"></a>–†–∞–≤–Ω–æ–º–µ—Ä–Ω–∞—è –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ—Å—Ç—å, –ø—Ä–∏–º–µ—Ä—ã

 $\forall \mathcal E > 0\ 
\exists \sigma\ 
\forall x_1, x_2\in E:\

–ì–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–∏–π —Å–º—ã—Å–ª —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–π –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ—Å—Ç–∏:

![–ö–∞—Ä—Ç–∏–Ω–∫–∞ —á–µ—Å—Ç–Ω–æ —Å—Ç—ã—Ä–µ–Ω–∞](math-exam-sem-2-2024/Untitled%202.png)

–ö–∞—Ä—Ç–∏–Ω–∫–∞ —á–µ—Å—Ç–Ω–æ —Å—Ç—ã—Ä–µ–Ω–∞

–ü—Ä–∏–º–µ—Ä: $f(x) = x^2$ –Ω–µ —è–≤–ª—è–µ—Ç—Å—è —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–π –Ω–∞ $\mathbb{R}$.

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º $\mathcal E = 1,\ 
x_1 = \frac 1 \sigma + \frac \sigma 2,\ 
x_2 = \frac 1 \sigma$,

$|x_1-x_2| =
\frac 1 \sigma + \frac \sigma 2 - \frac 1 \sigma =
\frac \sigma 2 < \sigma$,

–Ω–æ: $|x_1^2 - x_2^2| = 
|x_1-x_2|\cdot|x_1 + x_2| = 
\frac \sigma 2 \cdot (\frac 2 \sigma + \frac \sigma 2) =

–ù–∞ –ø–∞–ª—å—Ü–∞—Ö: –ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω–∏—á–µ–∫ –ø–æ —Ñ—É–Ω–∫—Ü–∏–∏ –¥–≤–∏–≥–∞—Ç—å –Ω–µ –ø–æ–ª—É—á–∏—Ç—Å—è, —Ç–∞–∫ –∫–∞–∫ –ø–∞—Ä–∞–±–æ–ª–∞ –≤–æ–∑—Ä–∞—Å—Ç–∞–µ—Ç —á–µ–º –¥–∞–ª—å—à–µ —Ç–µ–º –±—ã—Å—Ç—Ä–µ–µ.

**–¢–µ–æ—Ä–µ–º–∞ –ö–∞–Ω—Ç–æ—Ä–∞-–ì–µ–π–Ω–µ:** —Ñ—É–Ω–∫—Ü–∏—è, –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–∞—è –Ω–∞ –∑–∞–º–∫–Ω—É—Ç–æ–º –∫–æ–Ω–µ—á–Ω–æ–º –ø—Ä–æ–º–µ–∂—É—Ç–∫–µ (–∏–ª–∏ –Ω–∞ –ª—é–±–æ–º –∫–æ–º–ø–∞–∫—Ç–µ), —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–∞ –Ω–∞ –Ω—ë–º.

### <a name="1-15"></a>–ù–µ—Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–π –∏–Ω—Ç–µ–≥—Ä–∞–ª, —Å—Ç–µ–ø–µ–Ω–Ω–∞—è –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç—å

—Ç–æ–≥–¥–∞ –æ–ø—Ä–µ–¥–µ–ª–∏–º $\int\limits_a^\omega f(x)dx = \lim\limits_{b\to\omega}\int\limits_a^b f(x)dx$ ‚Äî **–Ω–µ—Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–π –∏–Ω—Ç–µ–≥—Ä–∞–ª**.

- –ï—Å–ª–∏ $\omega \in \mathbb{R}, f\in R[a,\omega]$, —Ç–æ –∑–Ω–∞—á–µ–Ω–∏—è –∏–Ω—Ç–µ—Ä–∞–ª–∞ –≤ —Ä–∏–º–∞–Ω–æ–≤–æ–º (—Å–æ–±—Å—Ç–≤–µ–Ω–Ω–æ–º) –∏ –Ω–µ—Å–æ–±—Å—Ç–≤–µ–Ω–Ω–æ–º —Å–º—ã—Å–ª–µ —Å–æ–≤–ø–∞–¥–∞—é—Ç.
- –õ–∏–Ω–µ–π–Ω–æ—Å—Ç—å

- –ü—É—Å—Ç—å $\phi:[a,\gamma) \to [a,\omega)$ ‚Äî –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä—É–µ–º–∞—è, —Å—Ç—Ä–æ–≥–æ –º–æ–Ω–æ—Ç–æ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è, $\phi(\alpha) = a$, $\lim\limits_{\beta\to\gamma} \phi(\beta) = \omega$, —Ç–æ–≥–¥–∞ $\int\limits_a^\omega f(x)dx =

### <a name="1-16"></a>–£—Å–ª–æ–≤–Ω–æ —Å—Ö–æ–¥—è—â–∏–π—Å—è –∏–Ω—Ç–µ–≥—Ä–∞–ª, –ø—Ä–∏–º–µ—Ä

–ü—Ä–∏–º–µ—Ä —É—Å–ª–æ–≤–Ω–æ —Å—Ö–æ–¥—è—â–µ–≥–æ—Å—è –∏–Ω—Ç–µ–≥—Ä–∞–ª–∞: $\int\limits_0^{\infty} \frac {\sin \alpha x\ dx} x =

### <a name="1-17"></a>–†—è–¥—ã, —Å—Ö–æ–¥–∏–º–æ—Å—Ç—å, –ø—Ä–∏–º–µ—Ä—ã

### <a name="1-18"></a>–ö–æ–Ω—Å—Ç–∞–Ω—Ç–∞ –≠–π–ª–µ—Ä–∞-–ú–∞—Å–∫–µ—Ä–æ–Ω–∏

**–ü–æ—Å—Ç–æ—è–Ω–Ω–∞—è –≠–π–ª–µ—Ä–∞-–ú–∞—Å–∫–µ—Ä–æ–Ω–∏** ‚Äî –ø—Ä–µ–¥–µ–ª —Ä–∞–∑–Ω–æ—Å—Ç–∏ –º–µ–∂–¥—É —á–∞—Å—Ç–∏—á–Ω–æ–π —Å—É–º–º–æ–π –≥–∞—Ä–º–æ–Ω–∏—á–µ—Å–∫–æ–≥–æ —Ä—è–¥–∞ –∏ –Ω–∞—Ç—É—Ä–∞–ª—å–Ω—ã–º –ª–æ–≥–∞—Ä–∏—Ñ–º–æ–º —á–∏—Å–ª–∞, —Ç–æ –µ—Å—Ç—å:

### <a name="1-19"></a>–û—Ä—Ç–æ–Ω–æ—Ä–º–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å–∏—Å—Ç–µ–º–∞ —Ñ—É–Ω–∫—Ü–∏–π

–µ—Å–ª–∏ —Ñ—É–Ω–∫—Ü–∏–∏ —Å–∏—Å—Ç–µ–º—ã –ø–æ–ø–∞—Ä–Ω–æ –æ—Ä—Ç–æ–≥–æ–Ω–∞–ª—å–Ω—ã –∏ –Ω–æ—Ä–º–∞ –∫–∞–∂–¥–æ–π –∏–∑ —Ñ—É–Ω–∫—Ü–∏–π —Ä–∞–≤–Ω–∞ $1$, —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ:  $\int\limits_a^b\phi_i(x)\phi_j(x)dx = \begin{cases}

### <a name="1-20"></a>–†—è–¥ –§—É—Ä—å–µ, –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã, –∫–æ–º–ø–ª–µ–∫—Å–Ω—ã–π –≤–∏–¥
